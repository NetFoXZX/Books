# Yandex Books Downloader - Расширение для браузера

Это простое расширение для Yandex Browser (Chrome) для скачивания книг с books.yandex.ru в форматах EPUB и аудиокниги.

## Особенности

- 📥 Скачивание книг в формате EPUB
- 🎧 Скачивание аудиокниг (ZIP архив)
- 🔄 Автоматическое использование текущей сессии браузера
- 📖 Поддержка всех книг из вашей библиотеки
- ⚡ Быстрое скачивание и сборка

## Установка

### Способ 1: Разработка (для тестирования)

0. Скачайте https://github.com/NetFoXZX/Books/archive/refs/heads/master.zip и распакуйте в папку `YaBooksDownloader`
1. Откройте Yandex Browser (или Chrome)
2. Перейдите в `chrome://extensions/` (или `browser://extensions/`)
3. Включите "Режим разработчика" (Developer mode)
4. Нажмите "Загрузить распакованное расширение" (Load unpacked)
5. Выберите папку `YaBooksDownloader`

## Использование

### Для электронных книг:

1. Откройте книгу на [books.yandex.ru](https://books.yandex.ru)
2. Нажмите на значок расширения в панели инструментов <img width="36" height="34" alt="image" src="https://github.com/user-attachments/assets/88a1ed05-a4e8-43a0-a3ff-5720942f8e4a" />

3. Нажмите "Скачать EPUB" (на значке расширения появится анимация)
4. Дождитесь завершения скачивания (анимированый зленеый квдрат пропадет)

### Для аудиокниг:

1. Откройте аудиокнигу на [books.yandex.ru](https://books.yandex.ru)
2. Нажмите на значок расширения в панели инструментов
3. Нажмите "Скачать Audio"
4. Дождитесь завершения скачивания (ZIP архив с аудиофайлами)

Аудиокниги сохраняются в ZIP архиве со структурой:
- `0001_*.mp3` или `0001_*.ogg` - первая часть/глава
- `0002_*.mp3` или `0002_*.ogg` - вторая часть/глава
- и т.д.

Файлы именованы с ведущими нулями для правильной сортировки.

## Технические детали

### Как это работает

1. **Content Script** (`content.js`) - внедряется на страницы reader и извлекает BookId из URL
2. **Popup** (`popup.html/js/css`) - интерфейс для управления скачиванием
3. **Background Script** (`background.js`) - основная логика:
   - Получает Session_id cookie из браузера
   - Запрашивает секрет для расшифровки
   - Загружает и расшифровывает метаданные книги
   - Скачивает контент (HTML, CSS, изображения)
   - Создаёт EPUB файл (ZIP архив)
   - Запускает скачивание через Chrome Downloads API

### API Yandex Books

- `GET /reader/p/api/v5/metadata_secret` - получение секрета для расшифровки
- `GET /p/api/v5/books/{bookId}/metadata/v4` - получение зашифрованных метаданных
- `GET /p/a/4/d/{uuid}/contents/OEBPS/{file}` - получение контента книги

### Формат EPUB

EPUB - это ZIP архив со специальной структурой:
- `mimetype` - тип контента (без сжатия)
- `META-INF/container.xml` - путь к package файлу
- `OEBPS/content.opf` - метаданные и манифест
- `OEBPS/toc.ncx` - оглавление
- `OEBPS/*.html` - контент книги
